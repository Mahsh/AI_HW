## 模板测试结论
1. 变量替换逻辑通顺，所有{{变量}}占位符均可一键替换；
2. 必填变量缺失会直接导致周报无核心数据源；
3. 数组类业务重点变量可自动同步至下周计划板块，变量设计合理；
4. 可选变量output_limit、last_week_risk不填时使用默认值，兼容性良好。
