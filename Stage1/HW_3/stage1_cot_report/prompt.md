# 内容安全审核团队周报生成提示词
## 分步指令要求
请严格按照以下4个步骤处理附件中的工作记录碎片，每一步都要显式输出结果，再进行下一步：

### 步骤1：主题分类
通读附件中2026年6月第3周的内容安全审核团队工作记录，将所有信息按「审核工单」「会议纪要」「策略更新」「异常事件」「其他（含个人TODO/备注）」5个主题分类整理，确保无信息遗漏。

### 步骤2：关键信息归纳
针对步骤1的每个分类，分别归纳「关键产出」（完成的动作、落地的方案等）和「关键数据」（可量化的数值、比例、时效等）。

### 步骤3：跨周风险项识别
识别本周未解决、需要下周继续跟进的风险/待办事项，标注事项类型、当前状态、潜在影响。

### 步骤4：生成结构化周报
按照以下格式生成最终周报，要求语言简洁、数据准确、逻辑清晰：
- 📌 本周概要（≤100字）
- 📊 关键数据（审核量 / 通过率 / 异常数 / 响应时效等）
- ✅ 本周完成（按主题分类，每条 ≤30字）
- 🔄 进行中（标注完成度和阻塞点）
- ⚠️ 风险与问题
- 📅 下周计划

## 附件说明
附件为《内容安全审核团队 · 工作记录碎片（2026年6月第3周）》，包含审核工单、会议、策略、异常事件、个人TODO等全量信息，请基于此完成所有步骤。
